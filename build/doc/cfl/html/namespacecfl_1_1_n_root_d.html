cfl::RootD cfl::NRootD::newton ( double  dAbsErr,
double  dRelErr,
unsigned  iMaxSteps = IMAX 
)

The classical Newton algorithm of polishing the roots of the function. We achieve the desired accuracy and stop as soon as

\[ |x_1-x_0| < \epsilon + \delta |x_1|, \]

where \(x_0\) and \(x_1\) are the previous and the current root estimates. We stop after iMaxSteps even if the desired accuracy has not been reached.

Parameters
dAbsErr( \(\epsilon\)) The absolute error.
dRelErr( \(\delta\)) The relative error.
iMaxStepsThe maximal number of iterations.
Returns
