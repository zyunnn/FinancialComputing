Classical least square fit of one-dimensional function \( f=f(x) \) given a basis function \(g=g(x)\) and a free function \(h=h(x)\). The family of fitting functions has the form:

\[ f(x,c) = cg(x) + h(x), \]

where \(c\) is the fitting coefficient.

Parameters
rBasisFThe basis function \(g=g(x)\).
rFreeFThe free function \(h=h(x)\).

Classical least square fit of one-dimensional function \( f=f(x) \) given a constant, a basis function \(g=g(x)\), and a free function \(h=h(x)\). The family of fitting functions has the form:

\[ f(x,c_0,c_1) = c_0 + c_1 g(x) + h(x), \]

where \((c_0,c_1)\) are the fitting coefficients.

rBasisFThe basis function \(g=g(x)\).
