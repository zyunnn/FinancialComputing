Detailed Description

Concrete class for one-dimensional fitting.

This is the standard class for one-dimensional fitting. It is implemented by a new instance of IFit.

Constructs the fitted one-dimensional function given the vectors of arguments and values. The vectors have identical sizes. Arguments are strictly increasing. The function is equivalent to the call of

cfl::Fit::assign(itArgBegin, itArgEnd, itValBegin, itWt, true)
cfl::Fit::assign
void assign(InIt1 itArgBegin, InIt1 itArgEnd, InIt2 itValBegin)

where all weights equal to 1. The normalizing standard deviation \(\kappa\) is obtained by \(\chi^2\) estimate.

itArgEndAn iterator to the end of the container of arguments.
itValBeginAn iterator to the start of the container of values.
