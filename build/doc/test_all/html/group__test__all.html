◆ sizeApprox()

std::function< unsigned(double)> test::sizeApprox ( double  dQuality)

Returns the number of nodes for approximation as a function of the interval \(dX\) using the formula:

  1. returns 1 if \(dX=0\);
  2. returns the ceiling of \( 2.5 + (1 + \sqrt{dQ}) dX / (1 + dX) \) if \(dX>0\).
Parameters
dQuality( \(dQ\)) The quality parameter.
Returns
std::function<unsigned(double)> The number of nodes for approximation as a function of the interval.
